# MineSweeper
This project is a Minesweeper built using Jetpack Compose.


# Requirements
Latest Android Studio Arctic fox and higher.
You can download it from [this page](https://developer.android.com/studio).


# Tech Stacks
- Jetpack Compose
- Dagger Hilt
- Coroutine & Flow
- ViewModel
